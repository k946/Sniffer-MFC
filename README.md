# Sniffer-MFC 1.0
基于winpcap的MFC窗口应用嗅探软件
## 功能：
1.0：   
  嗅探捕获数据包   
  解析数据包的头部（可解析的协议：ETHERNET,802.3,IP,ICMP,TCP,UDP,ARP）  
  编辑数据包  
  发送数据包（还没有重新计算验证和的功能）
